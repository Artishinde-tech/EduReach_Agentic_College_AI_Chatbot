# EduReach_Agentic_College_AI_Chatbot
EduReach Agentic College AI Chatbot is a GenAI-powered assistant built using LLMs, LangChain, and RAG. It provides accurate college-related information through text and voice interactions, leveraging semantic search, vector embeddings, and agentic workflows. Features a full-stack architecture with intelligent retrieval and conversational AI.
